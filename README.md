# Play 项目

这是一个基于 Python 的 Web UI 项目，集成了多种功能模块。

## 功能特性

- Web UI 界面
- 数据采集和处理
- 自动化工具集成
- 多种第三方库支持

## 技术栈

- Python 3.x
- FastAPI
- Uvicorn
- Playwright
- 多种数据处理库

## 快速开始

### 环境要求

- Python 3.7+
- Git

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动服务

```bash
# 使用批处理文件启动
run_webui.bat

# 或直接使用 Python
python web_ui.py
```

### 上传到 GitHub

使用提供的批处理文件上传代码：

```bash
upload_to_github.bat
```
